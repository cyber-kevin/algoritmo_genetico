# Algoritmo Genético

* As matrizes lidas pelo algoritmo devem ter o mesmo formato que as matrizes .txt anexadas para teste, caso contrário, o algoritmo poderá "quebrar".

* Esse algoritmo segue  a concepção de algoritmo genético e suas funções, como: gerar uma população de indivíduos, avaliar, selecionar os melhores indivíduos, cruzar, verificar possibilidade de mutação e repetir o processo. 

* Esse projeto foi criado para resolver um desafio de um food delivery fictício chamado "FlyFood", que realiza entregas através de drones. O principal objetivo deste trabalho foi de receber uma matriz de coordenadas e buscar uma rota com o menor percurso possível, a fim de minimizar gastos e recursos durante o serviço.
